Introduction

While working at MIM Software, a project opportunity arose to create a Scientific Poster Template for the Clinical Science team. Visual Designer, Nick Labate, and I collaborated in Figma to design the template and an accompanying tutorial.

Design System

To begin the project, I developed a component design system for the basis of the template in Figma. Below is a screenshot of the entire design system.

I broke it down from the smallest parts and eventually began to make larger components out of them. For example, I made a text component representing the different types of text that would be used in the template (headers, subtitles, body copy, captions, etc). Next I created a base section which included a title, body section, image, and caption. I knew most science posters were organized into columns, so then I created a column component that contained 3 sections- the number could be adjusted based on the user's needs. I also noticed sometimes posters would have rows so I created a row component containing 3 columns. Finally I created a header component and the final Poster Template component with variants based on the number of rows.

This design system allows for a multitude of responsive layouts to be created simply through selecting different options in the component's menu. It allows for non-designers to easily approach the template, create the layout needed for their content, and input their content without needing to worry about any of the design aesthetics or responsiveness.

The Template

Below are a couple different example layouts that can be created using the template component. Notice how unqiue each layout may be while also remaining consistent with style and format. The hope is that the flexibility and responsiveness of these templates will allow posters to be created and reformated based on specific needs.

Interactive Tutorial

After creating the template, Nick and I decided to work on creating a tutorial to help the Clinical Science team understand how to use the template. No one on the team had experience using Figma, so we had to cover the basics of Figma as well as how to use the poster template.

I came up with the idea to make an interactive tutorial, where individuals could follow along within Figma and make real-time edits to familiarize themselves with the software. I was inspired by Figma's own interactive tutorials. The material covered in the interactive tutorial broke down the basics of Figma, it's menus, and it's tools as well as an additional section on how to use the specific poster template. Overall, we recieved a positive response from those who followed along.

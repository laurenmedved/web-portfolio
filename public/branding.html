Purple Dog:

Purple dog is a project that originates from my Graphic Design 2 class at Baldwin Wallace University.

Each student was given a random company name and a random website type. I was given Purple Dog with a Portfolio website.
I decided to make Purple Dog a gaming studio that focused on retro-styled games. In the website mockup, I used my own games for content.
